# 更新紀錄

## 2026-07-17（115.07.17）網站重構＋三份官方清單整合

### 網站架構變更

原本為單一 `index.html`（403KB，資料內嵌），重構為：

```
index.html（27KB，僅 UI 與查詢邏輯）
├── data/products.js   下架產品資料（DATA 440 項＋TSDATA 泰山 42 項）
├── data/vendors.js    下游業者合併資料（VCITY／VDATA）
├── sources/           官方原始 PDF ×3
├── tools/             解析與建置腳本（python + pdfplumber）
└── archive/           重構前舊版 index.html 備份
```

重構理由：資料持續增長（本次業者紀錄倍增至萬筆級），資料與程式分離後，
之後更新只需重新產生 `data/*.js`，不動網頁本體；並保留原始 PDF 與解析
腳本，任何人可驗證資料來源與重現產出（`tools/` 腳本重跑輸出位元組一致）。
