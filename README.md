# 中聯油脂案｜下架產品與下游業者查詢

台灣 2026 年中聯油脂苯駢芘（BaP）超標事件的公開查詢網站。
純靜態網站，可直接部署於 **GitHub Pages**，也可以雙擊 `index.html` 離線使用。

## 目錄結構

```
checkU/
├── index.html          ← 網頁（UI 與查詢邏輯，不含資料）
├── data/
│   ├── products.js     ← const DATA（440項）+ const TSDATA（泰山42項）
│   └── vendors.js      ← const VCITY / VDATA（合併後業者資料）
├── sources/            ← 官方原始 PDF（供查證）
├── tools/              ← 資料解析與建置腳本（python + pdfplumber）
│   ├── parse_pdfs.py   ← sources/*.pdf → tools/parsed/*.json
│   ├── build_site.py   ← 合併資料 → data/*.js + index.html
│   └── index.template.html
├── archive/            ← 舊版網頁備份
└── README.md
```

## 部署到 GitHub Pages

1. 整個資料夾推上 GitHub（repo 根目錄需含 `index.html` 與 `data/`）。
2. Settings → Pages → Source 選 `main` branch、`/ (root)`。
3. 完成。純靜態、無任何後端與外部 API（僅 Google Fonts）。

## 之後官方公布新清單時如何更新

1. 把新 PDF 放進 `sources/`。
2. 在 `tools/parse_pdfs.py` 加上該 PDF 的解析（表格式清單可直接套用 `parse_simple()`），執行後產生 JSON。
3. 在 `tools/build_site.py` 的合併段落用新的來源 bitmask（8、16…）呼叫 `add_list()`，並在 `index.template.html` 的來源下拉選單與說明區補上新來源，重新執行即可產出新的 `data/*.js` 與 `index.html`。

環境需求：`python3 -m venv venv && venv/bin/pip install pdfplumber`

## 已知資料備註

- 批號欄出現「2.02704e+13」等字樣者，為官方原始檔的數字格式異常，照原樣呈現。
- 福懋清單含「零售」（未註明縣市）與「外銷」（IS(出口)／SOUTH KOREA）紀錄。
- 泰山名單之「產品毛重/淨重」欄位因原始 PDF 跨行嚴重、解析不可靠，未收錄。
- 泰山名單涉及之中聯批次含 316-1150504、316-1150515、318-1150517 等「7批」以外批號（屬 29 批全面預防性下架範圍）。
- 「新竹」「嘉義」未註明市/縣者，依原始清單照列。
